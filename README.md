# Playwright Automation Pro

## 🚀 Überblick
Dieses Projekt ist ein professionelles Playwright‑Automationsframework in Python.  
Es demonstriert moderne Testautomatisierung mit Page Object Model, strukturierten Test-Suites, Logging, Reports und erweiterbarer Architektur.

Das Ziel ist ein skalierbares Framework, das in realen QA‑Teams eingesetzt werden kann.

---

## 🧰 Technologien & Tools
- **Python**
- **Playwright**
- **Pytest**
- **Page Object Model (POM)**
- **Pytest Fixtures**
- **Allure Reporting** (optional erweiterbar)
- **GitHub Actions** (CI/CD vorbereitet)
- **Logging & Reports**

---

## 📂 Projektstruktur
Playwright_Automation_Pro/
│
├── Pages/                 # Page Object Dateien
├── Tests/                 # Testfälle (UI, Smoke, Regression)
├── Utilities/             # Helper, Logger, Config-Tools
├── reports/               # Testreports (HTML, Logs)
├── logs/                  # Laufzeit-Logs
│
├── conftest.py            # Browser- und Kontext-FIxtures
├── pytest.ini             # Pytest-Konfiguration
├── requirements.txt       # Python-Abhängigkeiten
└── .github/workflows/     # CI/CD Workflows (GitHub Actions
